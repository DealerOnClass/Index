Tools

Notepad, Browser

Objective

Become comfortable marking up text with the proper semantic typographic elements.

Instructions
  1. Open up notepad or similar text editor. NOT CODE EDITOR
  2. Create a new file with a .html extension (show hidden file extension in windows first).
  3. Grab dummy text from Fillerati.
  4. Markup the text using only typographic elements. No sectioning elements, or document structure.
  5. Save and open in browser.
