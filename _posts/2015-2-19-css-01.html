CSS

CSS is a set of overriding rules that define how HTML elements are displayed.

  • Cascading
  • Style
  • Sheets
Syntax

selector { property:value; }

p { font-size:12px; }

p {
font-size:12px;
line-height:1.4;
}

Selectors

Use CSS selectors to target HTML elements

Element

<h1 class="panel-heading">Chicken</h1>

h1 { font-size:12px; }

Class

<h1 class="panel-heading">Chicken</h1>

.panel-heading { font-size:12px; }

Id

<h1 id="panel-heading">Chicken</h1>

#panel-heading { font-size:12px; }

Descendant

<h1>
<span>Super</span>Chicken
</h1>

h1 span { color:blue; }

Direct Child

<h1>
<span>Super</span>Chicken
</h1>

h1 > span { color:blue; }

Adjacent

<h1>
<span>Super</span>Chicken
</h1>
<h1>
<span>Super</span>Chicken
</h1>

h1 + h1 { color:blue; }
