HTML5 Document

First we declare the <!DOCTYPE html> so the browser knows what it is reading, from there we wrap the entire document in an <html> tag.

Secondly, the meta information, title, internal stylesheets, and scripts will go within the <head;html> tag.

Finally, the actual page contents are contained within the <body> tag.

<!DOCTYPE html>
<html>
<head>
meta content…
</head>
<body>
body content…
</body>
</html>
