Float Property
The float property specifies whether or not a box element should float.

w3Schools

Block formatting context.

Remember
  1. Floated blocks display as inline-blocks
  2. The parent of a floated element requires special treatment*
    or…the next element in document flow must be cleared.
  3. Floats and widths are friends, along with box-sizing…*
  4. Inline elements can float.

* see below

.float--none {

}

Parent Cleared

Telling the div to hide overflowing elements is one of many techniques used to keep the parent of a floated element in document flow. Or you can use the clear property on the next element.

Check out other techniques here or simply Google "techniques for clearing floats".

.float--cleared {
overflow:hidden;
}
